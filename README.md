# Musicology Warehouse 🎵

## Overview
A PHP-based music inventory and e-commerce system that we created in our year 3 of Systems Development. Refactored to imporove slighly

## Features
- User authentication (Ajax)
- Cart system
- Album browsing
- Checkout system
- Search bar

### MySQL Integration
- Used MySQL to POST data to the DB's
- Used MySQL to Fetch data to the DB's

### CRUD Elements
- Cart system
- Add Albums to store
- Checkout system
- Stock Management
- Branch management
